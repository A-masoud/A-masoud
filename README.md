# مسعود علیپور

توسعه‌دهنده فرانت‌اند با تمرکز بر پیاده‌سازی وب‌اپلیکیشن‌های واقعی، ساخت‌یافته و قابل توسعه.
توانمند در تبدیل نیازمندی‌های محصول به رابط کاربری تمیز، سریع و مقیاس‌پذیر.

---

## خلاصه حرفه‌ای
- توسعه‌دهنده فرانت‌اند با تجربه عملی در پروژه‌های واقعی
- مسلط به Next.js 16 و TypeScript
- طراحی و پیاده‌سازی رابط کاربری مدرن با Tailwind CSS
- مدیریت وضعیت اپلیکیشن با Zustand
- پیاده‌سازی سیستم احراز هویت با JWT
- توانمند در کار با Git و GitHub در محیط‌های تیمی
- آشنا به Python برای پیاده‌سازی منطق برنامه و حل مسئله

## پروژه‌ها

### TommyShop | فروشگاه آنلاین
**تکنولوژی‌ها:**  
Next.js 16، TypeScript، Tailwind CSS، Zustand

**شرح پروژه:**
- طراحی و پیاده‌سازی سیستم ثبت‌نام و ورود کاربران با JWT
- پیاده‌سازی سبد خرید و مدیریت وضعیت آن
- صفحه جزئیات محصول با گالری تصاویر
- سیستم فیلتر، دسته‌بندی و جستجوی زنده با Debounce
- طراحی کاملاً ریسپانسیو و بهینه برای کاربر

مخزن پروژه:
