# مسعود علیپور

توسعه‌دهنده فرانت‌اند با تمرکز بر ساخت وب‌اپلیکیشن‌های واقعی، مقیاس‌پذیر و تمیز.
علاقه‌مند به معماری مناسب، رابط کاربری مدرن و یادگیری عمیق.

---

## درباره من
- تمرکز اصلی روی توسعه فرانت‌اند
- کار با Next.js 16 و TypeScript
- طراحی رابط کاربری با Tailwind CSS
- مدیریت وضعیت با Zustand
- پیاده‌سازی احراز هویت با JWT
- آشنا با Git و GitHub
- مسلط به Python در سطح متوسط تا بیشرفته
- دانشجوی مهندسی عمران، در حال تغییر مسیر حرفه‌ای به برنامه‌نویسی

---
